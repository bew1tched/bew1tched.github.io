Dokumentation

Aufgabenstellung

Erzeugen Sie mit Hilfe eine Grammatik zufällige arithmetische Ausdrücke über einstelligen natürlichen Zahlen mit den Operanden "+,-,*,/" und beliebig geschachtelten runden Klammeren, z.B.: 2*(3*(4+7)-9) oder 9+2 oder (3) oder nur 7 sind gültig, ungültig sind 12 oder () oder (+3) oder -1.
Nutzen Sie die folgenden Abkürzungen: A = Ausdruck, O = Operator = + | - | * | / , Z = Zahl/Ziffer = 0 | 1| 2| ... | 9.

Technische Umsetzung

Das Layout wurde grundsätzlich mit HTML5 und CSS3 erstellt.
Es handelt sich um eine Grid-View, in dem die einzelnen Bereiche aufgeteilt sind.
Das Design der Buttons und Textfelder wurde mit Bootstrap generiert.
Sämtliche Funktionalität wurde mit JavaScript implementiert.

Fachliche Umsetzung

Bei 'Klick' auf den Button "Generate Valid Expression" wird ein zufälliger valider Ausdruck erstellt.
Jeder Ausdruck beginnt immer mit S als Startwert. Aus S kann: SOS (ein Startwert mit Operator und noch ein Startwert), eine Zahl Z oder ein Ausdruck A folgen.
Es wird nun mit Hilfe von Math.random() ein zufälliger möglicher Nachfolger gewählt.
Gleiches gilt für O, wo ein zufälliger Operator oder Z, wo eine Zufällige Zahl ausgewählt wird.
Kommt es zur Auswahl A, dann wird der Ausdruck in Klammern geschrieben: (S).
Dies geschieht solange bis kein A, S oder Z mehr in dem so konstruierten String vorkommt.

Der Button "Generate Stepwise" funktioniert grundsätzlich nach dem selben Prinzip, nur muss manuel auf "Next" geklickt werden, bis ein terminaler Wert erreicht wurde.
Auch hier ist der String terminal, wenn kein A, S oder Z mehr vorkommt.

Der Button "Generate Invalid Expression" generiert wie bei "Generate Valid Expression" ersteinmal einen validen Ausdruck und löscht dann einen Random Wert wieder aus dem String.
Die Validierungsfunktion zählt nun erst alle Klammern um zu prüfen ob die Anzahl übereinstimmt. Außerdem werden alle Zahen einzeln in ein Array geschrieben und es wird geprüft, ob es Werte größer 9 gibt.
Zusätzlich wird notiert welches Zeichen an welcher Stelle entfernt wurde um später anzeigen zu können, warum der Ausdruck falsch ist.
